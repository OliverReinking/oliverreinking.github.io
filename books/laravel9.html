# Die ersten Schritte in Laravel 9

Laravel ist ein plattformübergreifendes PHP-Framework für die Entwicklung von Webanwendungen.

Laravel ermöglicht es einem Entwickler, auf eine unfassbare große Menge vorprogrammierter Funktionen zurückzugreifen. Robuste Webanwendungen sind schnell erstellt und gleichzeitig kann man den Umfang der notwendigen Codierung erheblich minimieren.

In diesem Buch wird eine Anwendungsplattform namens "Der Newspilot" entwickelt. Diese Anwendungsplattform besteht aus einer:

  • Homepage
  • einem Authentifzierungsverfahren
  • einer Anwendung für Administratoren
  • einer Anwendung für Mitarbeiter
  • einer Anwendung für Kunden.

Jeder Entwicklungsschritt ist im Detail dokumentiert. Neben Laravel 9 werden noch folgende "Techniken" verwendet:

  • Vue.js 3
  • Tailwind CSS 3
  • Inertia

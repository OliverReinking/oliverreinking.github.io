# CodingStarter

Der CodingStarter ist eine leistungsstarke Web-Anwendung, die mit dem Laravel-PHP-Framework entwickelt wurde und im Frontend Vue.js und Tailwind CSS verwendet. Im Backend kommen Inertia.js und Jetstream zum Einsatz. Die Anwendung besteht aus einer Homepage und separaten Anwendungen für Administratoren, Mitarbeiter und Kunden und ist damit der ideale Einstiegspunkt für die Erstellung neuer Web-Anwendungen.

Der CodingStarter dient in meinem Buch ChatGPT trifft Laravel als Startpunkt für die Entwicklung der Anwendung KreativSchreiber.
